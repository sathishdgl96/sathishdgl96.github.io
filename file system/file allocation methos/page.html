Contigous allocation:
All logical blocks of a file get the contigious physical block in Secondary storage
File system: Gives details about the file, when created, organizing

Advantages:
1. Fast Read
2. Supports Random Access

Disadvantages
1. Disk Fragementation
2. Difficult to increase the file size


Linked List Allocation:
Need not to be contiguos
Have file pointer

Advantages:

1, No external Fragementation

Disadvantages:
1. Random access not possible
2. pointer require space
3. pointer should not be broken
4. need to traverse all block

File Allocation Table:

There will be head block to point to start

Disadvantages:
1. Each disk blocj need fat entry



Indexed Allocation Scheme:
One block alloted to index all the block address

Advantages:
1. Support Direct access
2. If block damaged, only that block data will be lost

Disadvantage:
1. Bad Index block will result in entire file loss
2. Index block space is waste

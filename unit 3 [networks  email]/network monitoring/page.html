Levels of n/w monitoring:

According to IR, 4levels
1.Event based alerts [Intrusion detections]
2. Levels 2and 3= collecting packets and sessions [Header capture and full packet logging]
4.Statistical analysis => Results can be visualized [Statistical modelling ]

1. Event based alerting - can be helpful to identfy the affected computers quickly

2. Header captures - Can be helpful incase of data theft, useful to submit as evidence

4. statistical analysis can be helpful if we work on a large data



1. Event based alert monitoring:

=> usually done by IT professional, we will have set of rules or threshold
=> Whenever that crosses, it alerts
=> NIDS [Network intrusion detection system]
=> S/w Tools that supports event based monitoring
=> Tools:snort, suricate, RSA Netwitness [works based on signatures/ indicators]
- Can be simple [based on tcp packets] or complex rules [String matching, pattern matching]

2. Header and full packet logging:
=>To collect full data
=> Evidences to support for internal investigations

=> We are collecting everything[A- Z], Whenver you do perform analysis then remove un neccessary
=> Tcp dump, wireshark

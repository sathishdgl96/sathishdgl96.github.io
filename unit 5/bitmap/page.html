Computer files containing an image or pictures in digitial format

To store images:
1. so much information is required
2. pixels and color information
Data transmission requires bandwidth, To avoid we go with compression

Graphics files
1. bitmap images - combination of dots,Raster - collection of dots in row format
2. vector graphics- based on mathematical instructions [scaling won't cause pixelation]
3. metafile graphics - combination of bitmap and vector
eg: WMF - Windows meta files
  • EMF - Enhanced meta files

Types of programs
- Graphics Editors
- Image viewers

Bitmap images - Grids of individual pixels
Raster images - Pexels are stored in rows, Better for printing [Used in TV]

Image quality depends on
1. Screen Resolution
2. Software
3. Number of color bits used per pixel

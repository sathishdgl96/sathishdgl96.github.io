Windows Registry:
Config Data = OS, Application
Hives => System specific hives and user specific hives
User specific -> For a specific there will be one hive, User profile and history of activites
TimeStamp: Last write time [Specific for keys]

System hives: => 5 Hives
1. System
2. Security
3. SAM
4. Software
5. DEFAULT
No extension

User Hives
1. User Profile and User Activity
1. . DAT Extension
2. .\User\Usernam\NTUUser.DAT

Whenever user is created, the hive will be created. If the attack is happeneing over the network, analysing the user hives may not be beneficial

Tools for registry analyzer
1. RegRipper
2. Window Registry decoder
3. AutoRun
4. RedLine

Hive contains Key, value, date
Every key will have subkeys

ROOT, Current_User, HKEY_USERS
